# NOIRÉ — Digital Perfume Gallery

A minimal, premium hero section concept for a luxury fragrance online store. Built strictly with pure HTML and CSS, adhering to an elegant, dark aesthetic.

## Features

- **Pure HTML & CSS:** No JavaScript, no frameworks (React, Vue, etc.), and no CSS libraries (Tailwind, Bootstrap).
- **Dark Luxury Aesthetic:** Deep blacks, subtle gold accents, and stark white typography to emulate high-end fashion and fragrance brands.
- **Asymmetrical Editorial Layout:** Large serif typography paired with clean sans-serif secondary text, emphasizing negative space.
- **CSS-Only Visual:** A custom-designed perfume bottle created entirely using CSS shapes, gradients, and box-shadows to simulate glass and liquid reflections.
- **Smooth Animations:** Elegant fade-up animations for text elements and subtle hover interactions for the CTA and the bottle.
- **Fully Responsive:** Adapts seamlessly from large desktop displays down to mobile screens.

## File Structure

- `index.html`: The semantic HTML structure of the hero section and minimal navigation.
- `styles.css`: All styling, animations, responsive media queries, and the CSS bottle construction.
- `README.md`: Project documentation.

## How to View

Simply open the `index.html` file in any modern web browser. No build steps or local servers are required.

## Design Concept

The design direction is inspired by a blend of Apple's minimalism, Tom Ford's dark luxury, Porsche's precision, and premium museum exhibitions. The goal is to create a digital space that feels like stepping into a high-end physical gallery.
